# Gemini Studio Nano Banana Pro Prompt Templates

## Thumbnail Generation Structure

Use this prompt structure for each thumbnail variation:

```
Create a high-impact YouTube thumbnail in [VARIATION STYLE] style:

**Text Overlay:** "[HOOK TEXT]"

**Style Requirements:**
- Dominant colors: [COLOR PALETTE]
- Typography: Large, bold, all caps for key words
- Layout: [LAYOUT DESCRIPTION]
- Mood: [MOOD/ENERGY]

**Subject:**
- The subject is a real estate investing educator and entrepreneur
- He must appear prominently in the thumbnail with clear facial features
- Use the provided reference image to ensure accurate facial representation
- Expression should convey [EXPRESSION]

**Technical Specs:**
- Aspect ratio: 16:9 (landscape)
- Resolution: 1280×720
- High contrast for small-screen readability
- NO channel name, watermarks, or branding text

**Important:** The text overlay "[HOOK TEXT]" must be clearly visible and readable at thumbnail size.
```
