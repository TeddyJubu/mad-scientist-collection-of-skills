# Daily End-of-Day Report Generator

**Goal:** Compile a professional PDF report of all agent activity for the day and send it via Telegram at 5 PM ET.

## Execution Steps (Cron Context Safe)

Since this runs in a fresh context with no session memory, use these methods to find activity:

### Step 1: Gather Data
1. **Check Recent Files:**
   ```bash
   # Check estimates and reports created today
   find /root/.hermes/ -type f -mtime 0 -name "*.md" -o -name "*.pdf" -o -name "*.html" 2>/dev/null
   ```
2. **Check Cron Logs for Runs:**
   ```bash
   # Check recent cron history
   ls -lt ~/.hermes/cron/output/ 2>/dev/null | head -20
   ```
3. **Use `session_search` (Best Effort):**
   Try to recall keywords like "deploy", "estimate", "report", "agent", "skill", "home depot", etc. It may return results if the system indexes them.
4. **Check Skill Modifications:**
   ```bash
   ls -lt /root/.hermes/skills/ 2>/dev/null | head -20
   ```

### Step 2: Build the Report
Create an HTML file with the structure:
- **Header:** Dark navy (#1a1a2e) with red (#e94560) accent border.
- **Executive Summary:** 2-3 sentences highlighting the biggest wins.
- **Agent Activity:** Cards for James, Sarah, Nova, Michael, Tammie, Eric, Bob, Mark.
  - If you find evidence of work (files, logs), list it.
  - If NO evidence, mark as "NO ACTIVITY".
- **System & Automation:**
  - Skills created/patched.
  - Cron jobs run status.
  - New files created.
- **Action Items / Notes:** Any issues found (e.g., failed jobs, errors).
- **Footer:** Generated by James, date.

### Step 3: Generate PDF
```bash
weasyprint /tmp/eod-report.html /tmp/eod-report.pdf
```

### Step 4: Deliver
Include `MEDIA:/tmp/eod-report.pdf` in the final Telegram message.

## Notes
- If `weasyprint` is missing, try `apt-get install weasyprint`.
- Always include the date in the report title: "End-of-Day Agent Report — [Date]".
- If absolutely no data is found, report "System Quiet - No activity recorded in available logs."
