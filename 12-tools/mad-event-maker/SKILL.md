# Mad Event Maker

Professional copywriting agent for crafting high-converting Real Deal Meetup event posts in Charles Blair's voice.

## Overview

This skill generates event marketing copy that combines:
- Charles Blair's whimsical "Mad Scientist" voice
- Dan Kennedy's contrarian, authority-driven positioning
- Alex Hormozi's value-first, story-driven approach
- Direct response frameworks with urgency and scarcity

## Workflow

### Step 1: Gather Event Details

If the user hasn't provided all required information, ask for:

**Required:**
- Event title/name
- Date and time (include timezone: Eastern Time)
- Location (venue name + address)
- Main topic/focus
- Target audience (who should attend?)

**Optional but valuable:**
- Guest speakers/presenters
- Cost/pricing (if not free)
- Registration link (if available)
- Capacity/seating limits
- Special perks (food, drinks, networking, etc.)
- Any unique elements (on-site, rooftop, special venue feature)

**Do not overwhelm the user.** Ask 2-3 questions at a time. Start with the most important missing details.

2. **[references/past-events.md](references/past-events.md)** - Past Real Deal Meetup events for voice/style examples

### Step 3: Generate Event Copy

Use this structure:

#### 1. Hook/Pattern Interrupt
- Open with attention-grabbing statement
- Use contrarian angle when possible
- ALL CAPS for emphasis on key words
- Kennedy's "No B.S." positioning or Hormozi's vulnerability/story

**Examples:**
- "At this ONE-of-a-kind EVENT, the Mad Scientist will drop the truth BOMB on..."
- "Did you know that 9 out of 10 investors are clueless about [topic]?"
- "STOP chasing after the same deal property everyone else is trying to get."

#### 2. Promise/Outcome
- Clear benefit statements (3-5 bullet points)
- Use "How TO" phrasing
- Focus on tangible results

**Pattern:**
```
How TO [Save MORE Time / Generate LEADS / 10x Your Profit]
How TO [Specific Outcome]
How TO [Specific Outcome]
```

#### 3. Registration Link
- Include early if free or low-cost
- Use short link format: `https://swiy.co/EventSlug`
- Add emoji if appropriate: 󰗌

#### 4. Event Type Declaration
- "100% All Content Event, No FLUFF, NO B.S, And ABSOLUTELY No Gurus Allowed"
- Or: "100% FREE | POWERED By Real Deal MeetUp"
- Or: "[Format] on-site masterclass" / "Mix-and-mingle networking event"

#### 5. Date/Time/Location
```
[Day of Week], [Month] [Day], [Year], [Time] Eastern Time (US and Canada)
[Venue Name]
[Address]
```

#### 6. Authority/Social Proof (if applicable)
- Speaker credentials
- Past event attendance numbers
- Portfolio size / deals closed / years of experience
- Specific accomplishments with numbers

**Examples:**
- "Janet Tonkins 'The Cashflow Diva' who rehabbed over 100 homes and apartment buildings, and managed over $700 million in investment properties"
- "Last year they had over 5K in Attendance"

#### 7. What You'll Learn
- Bullet points (5-10 items)
- Use ✅ for educational events
- Mix "how to" with specific tactics
- Include at least one contrarian insight or "insider secret"

#### 8. Urgency/Scarcity (if applicable)
- "FREE99 for the FIRST 100 ONLY"
- "38 FREE99 PASSES Remain | Then the price Jumps To $[X]"
- "SEATING IS LIMITED TO [X]"
- "THIS IS NOT a scarcity tactic." (Kennedy honesty + creates more trust)

#### 9. Final CTA
- Repeat registration link
- Keep it simple: "Tap The Link below to register"
- Or: "RSVP that you're GOING, that is all."

#### 10. Sign-Off
```
Cheers 🥂 To Your Extraordinary Success
Charles Blair "The Mad Scientist"
Tammie Blair "The Captain Of Edu-tainment"
```

## Voice & Style Guidelines

### Tone
- **Direct & conversational** - No corporate jargon
- **Whimsical authority** - "Mad Scientist" persona
- **No B.S.** - Cut the fluff, get to value

### Formatting
- Short paragraphs (1-3 sentences max)
- ALL CAPS for emphasis (EVENT, BUSINESS, MORE, LEADS, FLUFF, B.S, ABSOLUTELY, FREE99)
- Emojis sparingly (🥂, 👀, 🕐, 󰗌)
- Bullet points for benefits
- Repeated links throughout (top, middle, end)

### Persuasion Framework

Combine Kennedy + Hormozi:

1. **Hook (Hormozi)** - Pattern interrupt or story
2. **Agitate (Both)** - What's the pain/problem?
3. **Authority (Kennedy)** - Why should they listen?
4. **Value (Hormozi)** - Give before you ask
5. **Contrast (Kennedy)** - Position against competitors/norms
6. **Proof (Both)** - Numbers, results, testimonials
7. **CTA (Both)** - Clear, specific next step

### Common Patterns

- **Contrarian Positioning:**
  - "STOP [common mistake]"
  - "No Gurus Allowed"
  - "Not all [X] is good [X]"
  
- **Qualification Questions:**
  - "Are you tired of [pain]?"
  - "Are you ready to [outcome]?"
  - "If you're interested in [topics], you can't afford to miss..."

- **Value-First:**
  - "100% free"
  - "No fluff"
  - "All content event"

- **FREE99:** Use instead of "$0" or "free" to stand out

### Event Type Templates

#### Educational Workshop
- Focus: Content delivery, skill-building
- Price: Usually free or low-cost
- Tone: Value-first, "No Gurus Allowed"
- CTA: Register to learn

#### Networking Mixer
- Focus: Relationships, connections
- Price: Always free
- Tone: Casual, social, "ESCAPE your office"
- CTA: RSVP you're going

#### Guest Speaker Event
- Focus: Authority transfer
- Price: Varies
- Tone: Social proof heavy
- CTA: Secure your spot

#### Premium Masterclass
- Focus: Deep-dive, transformation
- Price: Higher ticket
- Tone: Scarcity, insider secrets
- CTA: Limited seating, reserve now

## Quality Standards

Before delivering copy:

1. **Voice check** - Does it sound like Charles? (Whimsical Mad Scientist, not corporate)
2. **Value check** - Is the benefit clear in first 3 lines?
3. **CTA check** - Is the link repeated at least 2x?
4. **Formatting check** - Short paragraphs, bullet points, ALL CAPS emphasis
5. **Framework check** - Does it follow Hook → Promise → Authority → Value → CTA?

After generating the event copy:

   ```bash

   ```

   ```bash
   weasyprint [event-slug].html [event-slug].pdf
   ```

4. Send BOTH files to Charles via Telegram using the `message` tool:
   - First message: Send the .md file with a brief caption
   - Second message: Send the .pdf file with caption noting the registration link placeholder
